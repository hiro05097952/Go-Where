# Go Where 電商

> 使用 Vue + Nuxt 打造的電商網站，主要商品為古著與鞋子。

## Demo

Nuxt 版本：[https://go-where-nuxt.herokuapp.com/](https://go-where-nuxt.herokuapp.com/)

Vue 版本：[https://jedy05097952.github.io/Go-Where/#/](https://jedy05097952.github.io/Go-Where/#/)

## 簡介

此作品使用 Nuxt 打造，後端使用 Node.js 搭配資料庫 Firebase

### 前台

* 會員註冊 / 登入 ( 支持第三方 Google )

* 結帳 / 下單 ( 綠界金流 )

* 購物車

* 優惠卷

* 追蹤商品

* 會員頁面管理

### 後台

* 上 / 下架商品

* 管理優惠卷

* 訂單管理

* 模擬下單系統

## 使用技術

* Vue
  Vue Router
  Vuex
  Vue Cli 3

* Nuxt

* ESLint ( Airbnb )

* JavaScript

* Bootstrap4

* Scss

* Node.js

* Firebase
  Firestore
  Authentication
  Storage

* Git

## Build Setup

```bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
